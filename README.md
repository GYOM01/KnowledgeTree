# 知识树 Knowledge Tree v0.1

正式 Android 原生工程，技术栈：Kotlin + Jetpack Compose + Material 3。

## 当前完成
- 原生首页
- 搜索框与本地筛选
- 知识类型标签
- 能力型、历史型、技能型、分类型四种示例结构
- 节点详情与步骤列表
- 收藏、学习记录、设置页面入口
- GitHub Actions 自动构建 APK

## 构建
在 Android Studio 中打开项目并同步 Gradle，或上传到 GitHub 后运行 Actions → Build Android APK。

## 兼容
minSdk 26，兼容 Android 8.0 及以上；Pixel 2 XL 的 Android 11 可运行。
