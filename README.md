# 知识树 Knowledge Tree v0.2

Android 原生工程：Kotlin + Jetpack Compose + Material 3。

## v0.2 构建修复
- 固定 Android Gradle Plugin 8.7.3
- 固定 Kotlin / Compose Compiler 2.1.0
- 固定 Gradle 8.9
- 使用 compileSdk / targetSdk 35
- GitHub Actions 自动安装 Android SDK 35
- 自动生成并上传 debug APK Artifact

## 获取 APK
进入 GitHub 仓库 Actions → Build Android APK，等待绿色对勾，在运行详情底部下载 `KnowledgeTree-v0.2-debug-apk`。
